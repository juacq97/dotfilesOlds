Im Sinne
[[https://de.wikipedia.org/wiki/Open_Educational_Resources][offener Bildungsmaterialien]]
(engl. Open Educational Resources, OER) veröffentliche ich
{{{gitlablink(Quelldateien\, benötigte Software und Präsentationen in diesem GitLab-Repository)}}}
unter
[[https://de.wikipedia.org/wiki/Freie_Lizenz][freiheitsgewährenden Lizenzen]].
Alle OER-Präsentationen werden aus Textdateien in
einem einfachen Textformat namens [[https://orgmode.org/][Org Mode]]
erzeugt; dies erlaubt den Fokus auf Inhalte, während das Layout
separat definiert wird.  Diese Trennung von Inhalt und Layout
vereinfacht die Zusammenarbeit über Organisationsgrenzen hinweg, und
die Verwendung eines einfachen Textformats ermöglicht Vergleiche von
geänderten Versionen (mit ~diff~-artiger Funktionalität).

Die freie Software
[[https://gitlab.com/oer/emacs-reveal][emacs-reveal]] überführt
die Textdateien in
[[https://revealjs.com/][reveal.js]]-HTML-Präsentationen,
die auf (nahezu) jedem Gerät mit Web-Browser angesehen werden können.
In Zeiten von Massenüberwachung und Überwachungskapitalismus
[[https://www.informationelle-selbstbestimmung-im-internet.de/][empfehle ich die Firefox-Variante Tor Browser]]
als digitale Selbstverteidigung; die OER-Präsentationen funktionieren
auch unter der verschärften Sicherheitseinstellung „Safer“ im Tor
Browser.
